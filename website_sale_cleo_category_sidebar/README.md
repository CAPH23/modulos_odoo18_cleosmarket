# Super Tienda Cleo - Menú lateral de categorías

Módulo independiente para Odoo 18 que rediseña el menú lateral de categorías de `/shop`.

## Características

- Usa `product.public.category`.
- Enlaces estándar `/shop/category/<slug>`.
- Excluye CANASTAS, CELOFAN, CELOFÁN, PRODUCTOS PLÁSTICOS y PRODUCTOS PLASTICOS.
- Botón colapsable.
- Estado colapsado guardado en `localStorage`.
- `position: sticky` en escritorio.
- Icono del módulo basado en el logo de Super Tienda Cleo.
- Diseño visual crema, azul marino y amarillo, inspirado en la propuesta aprobada.
